GlobalBridge AI

Sovereign real-time translation — 100+ languages, live captions, Whisper dictation. Built on Tether QVAC + Keet P2P.

Download for Windows Install guide Watch demo GitHub

Get started (2 steps)

1

Install GlobalBridge on your PC

Download the Windows installer from GitHub Releases, run Setup from the tray (Python 3.11+ required once), then Start services.

2

Add the Chrome extension

Install GlobalBridge Live Captions from the Chrome Web Store (listing pending Google review — use Load unpacked from the repo extension/ folder until then).

Open YouTube or Meet → extension icon → Start captions.

Live captions

Chrome tab audio → local Whisper → QVAC translation. YouTube, Zoom, Meet, Teams.

Keet meetings

P2P encrypted calls with personalized bilingual subtitles. Export TXT / JSON / SRT.

Sovereign Mode

No audio or transcript egress to third-party APIs when local processing is enabled.

Developer install

git clone https://github.com/001453/newtranslate.git
cd newtranslate
npm run setup
npm run dev

Open http://localhost:3000 — requires QVAC (:8765), API (:8000), Chrome for tab audio.
