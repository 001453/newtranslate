GlobalBridge AI

Sovereign real-time translation — 100+ languages, live captions, Whisper dictation. Built on Tether QVAC + Keet P2P.

View on GitHub Download v0.1.0 Quick Start

Live captions

Chrome tab audio → local Whisper → QVAC translation. YouTube, Zoom, Meet, Teams.

Keet meetings

P2P encrypted calls with personalized bilingual subtitles. Export TXT / JSON / SRT.

Sovereign Mode

No audio or transcript egress to third-party APIs when local processing is enabled.

Quick install

git clone https://github.com/001453/newtranslate.git
cd newtranslate
npm run setup
npm run dev

Open http://localhost:3000 — requires QVAC (:8765), API (:8000), Chrome for tab audio.
