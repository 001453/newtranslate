Install Chrome Extension

Live captions on any tab (YouTube, Meet, Keet). Requires GlobalBridge services running.

  1. Open chrome://extensions
  2. Enable Developer mode (top right)
  3. Click Load unpacked
  4. Select the folder below (or the extracted zip)
  5. Pin GlobalBridge Live Captions to the toolbar
  6. Open a tab with speech → extension icon → Start captions
…

Chrome Web Store: Zip the extension folder and submit at Chrome Web Store Developer Dashboard. See docs/DISTRIBUTION.md in the repo.
